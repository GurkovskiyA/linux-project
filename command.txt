- Показать текущую директорию [ pwd ]
- Показать все файлы с правами [ ls -la ]
- Перейти в директорию [ cd /путь ]
-  Создать директорию [ mkdir имя ]
-  Удалить рекурсивно [ rm -rf имя ]

#### 1. Основы работы с терминалом

- Навигация по файловой системе (`ls`, `cd`, `pwd`)
    
- Работа с файлами и папками (`touch`, `mkdir`, `rm`, `cp`, `mv`)
    
- Просмотр содержимого файлов (`cat`, `less`, `head`, `tail`)
    
- Поиск файлов (`find`, `locate`)
    

#### 2. Права доступа и владельцы файлов

- Понимание `rwx` прав
    
- Команды `chmod`, `chown`, `chgrp`
    
- Просмотр прав: `ls -l`
    

#### 3. Управление процессами

- Просмотр процессов: `ps`, `top`, `htop`
    
- Завершение процессов: `kill`, `killall`
    

#### 4. Работа с пакетами

- Установка, удаление, обновление ПО с `apt` (Ubuntu)
    
- Поиск пакетов
    

#### 5. Сетевые команды

- Проверка подключения: `ping`, `traceroute`
    
- Просмотр сетевых интерфейсов: `ifconfig` или `ip a`
    
- Проверка портов: `netstat`, `ss`
    

#### 6. Редакторы файлов

- Основы работы в `nano` или `vim` (лучше начать с nano — проще)
